# 🍹 The Chill Zone Cafe & Mocktail

A fully responsive cafe website built with React.js for **The Chill Zone Cafe & Mocktail**, Latur, Maharashtra.

🌐 **Live Demo:** [amanulla4.github.io/the-chill-zone](https://amanulla4.github.io/the-chill-zone/)

---

## ✨ Features

- 🏠 **Hero Section** — Full-screen cafe background with CTA
- 📖 **About Section** — Story and highlights of the cafe
- 🍕 **Menu Section** — Interactive category-wise menu with prices
- 📸 **Gallery Section** — Photo gallery with lightbox and category filters
- 🎉 **Events Section** — Birthday, group, date night packages
- ⭐ **Testimonials** — Customer reviews
- 📍 **Contact Section** — Location, hours, WhatsApp & Google Maps
- 🔗 **Footer** — Quick links and social media

---

## 🛠️ Tech Stack

- **React.js** — Component-based UI
- **CSS3** — Custom styling with animations
- **GitHub Pages** — Deployment

## 🚀 Getting Started

```bash
cd the-chill-zone
npm install
npm start
```
